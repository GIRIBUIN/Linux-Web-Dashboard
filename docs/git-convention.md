# Git Commit Convention

| Type | Description |
|------|-------------|
| feat | 새로운 기능에 대한 커밋 |
| fix | 버그 수정에 대한 커밋 |
| build | 빌드 관련 파일 수정 / 모듈 설치 또는 삭제 |
| chore | 그 외 자잘한 수정 |
| ci | CI 관련 설정 수정 |
| docs | 문서 수정 |
| style | 코드 스타일 혹은 포맷 수정 |
| refactor | 코드 리팩토링 |
| test | 테스트 코드 수정 |
| perf | 성능 개선 |
